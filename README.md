 💪 BMI Calculator

A simple and interactive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript. This project allows users to calculate their BMI with a clean UI and fun emoji-based feedback.

📌 Features

- 🎯 Calculate BMI instantly
- 📏 Accepts height in centimeters (cm)
- ⚖️ Accepts weight in kilograms (kg)
- 😎 Displays BMI category with emojis
- 🎨 Attractive and modern UI design
- ⚡ Fast and responsive
